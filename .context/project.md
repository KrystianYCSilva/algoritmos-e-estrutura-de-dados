# Project Overview

## What

Biblioteca acadêmica em C11 implementando estruturas de dados eficientes, algoritmos clássicos, heurísticas e meta-heurísticas com rigor acadêmico e referências científicas.

## Why

Fornecer implementações de referência para ensino e pesquisa em algoritmos e estruturas de dados, com análise de complexidade completa e suporte a tipos genéricos em C puro.

## Scope

**In scope:**
- Estruturas de dados fundamentais (Fase 1: Queue, Stack, List, Hash, Tree, Graph, etc.)
- Algoritmos clássicos (sorting, searching, graph algorithms)
- Heurísticas construtivas e meta-heurísticas
- Análise de complexidade teórica e empírica
- Documentação acadêmica com referências

**Out of scope:**
- Estruturas de dados concorrentes
- Otimizações específicas de hardware
- Interfaces gráficas

## Status

**Phase 1A - COMPLETE:** Queue, Stack, LinkedList, ArrayList (com testes completos)  
**Phase 1B - IN PROGRESS:** HashTable ✓, BinaryTree ✓, BST, Heap, Graph  
**Phase 1C - PLANNED:** AVL Tree, Priority Queue, Trie, Union-Find
